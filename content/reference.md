## UART Signal Description

| Signal | Description         |
| ------ | ------------------- |
| CI     | Command Input       |
| CO     | Command Output      |
| RST    | Reset signal        |
| CLKI   | Clock Input         |
| CLKO   | Clock Output        |
| RI     | Respond Input       |
| RO     | Respond Output      |
| BI     | Respond Busy Input  |
| BO     | Respond Busy Output |
